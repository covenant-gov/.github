# Pacto Community Health Files

Default community health files for the [Pacto](https://github.com/covenant-gov) project. These files apply to all repos in the org unless a repo provides its own.

## Files

This repo is under construction. These files are placeholders, and will be replaced with Pacto's official rules as they are co-created by the crew.

| File | Purpose |
|------|---------|
| [GOVERNANCE.md](GOVERNANCE.md) | How decisions are made and authority is distributed |
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Expectations for members of Pacto's crew |
| [CONTRIBUTING.md](CONTRIBUTING.md) | How to contribute, support channels, PR process |
| [SECURITY.md](SECURITY.md) | Vulnerability reporting policy |

## Placeholders

| Placeholder | File | When |
|-------------|------|------|
| `[discord-invite-link]` | CODE_OF_CONDUCT.md, CONTRIBUTING.md | Discord server created |
| `[security-email]` | SECURITY.md | Security email configured |
